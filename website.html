Learning process:

After finishing HTML courses of Code academy, we slightly started codding. Firstly, we brainstormed about content. Due to absence of our CSS information that time we only had ideas about what to write and where to write. After learning CSS, we figured out how to design a website. However, selection of colors and how to and where to put elements was complicated as well, owing to the fact that we were looking for the perfect design with ideal color contrast. Additional to HTML and CSS we learned JavaScript as bonus, even though it is not used during creation of this website.


GitHub:

Since we were novice, and we it was first time that we would use GitHub, we did not know what to do there. After practicing in GitHub and learning some features of it from tutorial videos, we figured out how to work on it. GitHub Desktop is different to use from web version, therefore we put some effort on it as well.


Our logo and team name:

Our search for logo and team name took the most time. Because we could not find the best matching logo and team name. However, after surfing a lot on the internet, we could find a logo related to our project and liked by us. We chose our team’s name based on the logo as well.
